<objective>
Build Photophore's shadow generator and result-policy authoring module, plus the `photophore policy preview` CLI subcommand. Resolve OQ-2 (expose `ResultPolicy` as public from thermocline-py via a precursor spec patch), OQ-3 (shadow soft-fail return type = `ShadowResult(shadow, warnings)`), the shadow-strategy shape (closed enum + match), and the irreversibility threshold (8 chars). Ship the AT-A2 conformance fixture wired behaviorally via the SHADOW-02 uniqueness Hypothesis property test.

Purpose: Shadows are the dispatch-time abstraction that lets tier-1 content cross channel boundaries without leaking source bytes. SHADOW-06 (no caching) is the privacy guarantee: identical content dispatched twice produces two distinct shadow_ids, so an attacker correlating shadow_ids across boundaries cannot reconstruct content provenance. SHADOW-04 hard-fail (irreversibility) is the structural defense — abstractions that leak source substrings are rejected before being returned. POLICY-01/02 author the result_policy on the issuer node BEFORE signing, ignoring any policy in the draft (the draft cannot promote itself).

Output: `photophore.shadow` module with `generate(content, content_type, relevance) -> ShadowResult`, six per-content-type abstraction strategies, three quality tests; `photophore.policy` module with `author(channel, envelope_draft) -> ResultPolicy`; `photophore policy preview` CLI subcommand; AT-A2 conformance fixture; thermocline-py spec patch exposing `ResultPolicy` publicly; Hypothesis property test (>=100 cases) for shadow_id uniqueness.
</objective>

**`src/photophore/shadow/_strategies.py`** — Per-content-type abstraction strategies (closed enum + match):
```python
"""Per-content-type abstraction strategies per Photophore v0.3 §'Shadow Generation Quality'.

Each strategy must include the MUST signals and exclude the MUST NOT signals listed
in the spec table. Closed enum + match pattern (planner-resolved per CONTEXT.md
Claude's Discretion). Adding a 7th content type in v0.2 = add enum member + match arm.

Strategy implementations are intentionally simple — they generate generic templated
abstractions, NOT verbatim quotes. The irreversibility test (in _quality.py) catches
any leakage; the strategies are constructed to avoid it.
"""
from __future__ import annotations
from ._types import ContentType

def _length_class(content: bytes) -> str:
    n = len(content)
    if n < 1000:
        return "short"
    if n < 50_000:
        return "medium"
    return "long"

def _abstract_document(content: bytes) -> str:
    """MUST: topic category, length class, temporal indicator.
       MUST NOT: filename, author, specific dates, org names, unique IDs.
    """
    # v0.1: minimal heuristic — topic category is "general"; temporal is "current".
    return f"document of length class {_length_class(content)}, topic category general, temporal current"

def _abstract_conversation(content: bytes) -> str:
    """MUST: participant count, topic domain, tone.
       MUST NOT: participant names, quotes, specific claims, timestamps.
    """
    return f"conversation with multiple participants, topic domain general, tone neutral"

def _abstract_credential(content: bytes) -> str:
    """MUST: credential type label only.
       MUST NOT: credential value, service name, account identifier.
    """
    # v0.1: detect credential family via prefix; otherwise "generic credential".
    if content.startswith(b"-----BEGIN"):
        return "credential of type private-key"
    if content[:4] in (b"AKIA",) or any(content[:8].startswith(p) for p in (b"AKIA",)):
        return "credential of type cloud-api-key"
    if b"sk-" in content[:32]:
        return "credential of type api-token"
    return "credential of type generic"

def _abstract_file(content: bytes) -> str:
    """MUST: file type category, approx size class.
       MUST NOT: filename, path components, EXIF, embedded metadata.
    """
    return f"file of category data, size class {_length_class(content)}"

def _abstract_identity(content: bytes) -> str:
    """MUST: identity type only.
       MUST NOT: identity value, associated accounts, contact info.
    """
    return "identity of type person"

def _abstract_code(content: bytes) -> str:
    """MUST: language, complexity, domain.
       MUST NOT: repo name, function names, variable names, comments.
    """
    return f"code of complexity {_length_class(content)}, domain general"

def _generate_abstraction(content: bytes, content_type: ContentType) -> str:
    """Dispatch on content_type. Closed match — mypy --strict catches missing arms."""
    match content_type:
        case ContentType.DOCUMENT:
            return _abstract_document(content)
        case ContentType.CONVERSATION:
            return _abstract_conversation(content)
        case ContentType.CREDENTIAL:
            return _abstract_credential(content)
        case ContentType.FILE:
            return _abstract_file(content)
        case ContentType.IDENTITY:
            return _abstract_identity(content)
        case ContentType.CODE:
            return _abstract_code(content)
```

**`src/photophore/shadow/_quality.py`** — Three quality tests:
```python
"""Shadow quality tests per SHADOW-04.

Hard fail: irreversibility_test (raises ShadowIrreversibilityError) — abstraction leaks
source content substrings of length >= _IRREVERSIBILITY_MIN_SUBSTR_LEN.

Soft warn: relevance_preservation_test, distinguishability_test — return a list of
warning strings. Caller (generate()) collects warnings; Phase 3 dispatch records to audit.
"""
from __future__ import annotations
from ..errors import ShadowIrreversibilityError

# Pinned by 02-RESEARCH.md key finding §7. 4-char threshold produces false positives;
# 8-char threshold passes verified test cases. Named constant per CLASS-06 / SHADOW-04 idiom.
_IRREVERSIBILITY_MIN_SUBSTR_LEN: int = 8

def irreversibility_test(source_content: bytes, abstraction: str) -> None:
    """Raise ShadowIrreversibilityError if abstraction leaks any source substring
    of length >= _IRREVERSIBILITY_MIN_SUBSTR_LEN.

    Binary content (un-decodable UTF-8) cannot leak via substring check; return without
    raising in that case (defensive — invalid UTF-8 is not leakage).
    """
    try:
        source_text = source_content.decode("utf-8", errors="strict")
    except UnicodeDecodeError:
        return  # Binary content — substring check not applicable
    n = len(source_text)
    if n < _IRREVERSIBILITY_MIN_SUBSTR_LEN:
        return  # source too short to leak substrings of the threshold length
    for i in range(n - _IRREVERSIBILITY_MIN_SUBSTR_LEN + 1):
        substr = source_text[i:i + _IRREVERSIBILITY_MIN_SUBSTR_LEN]
        if substr.strip() and substr in abstraction:
            raise ShadowIrreversibilityError(
                f"abstraction leaks source substring {substr!r}",
                code="SHADOW_IRREVERSIBILITY_FAILED",
            )

def relevance_preservation_test(source_content: bytes, abstraction: str, relevance: float) -> list[str]:
    """SOFT WARN: return list of warning strings if abstraction's apparent information
    content is poorly matched to the declared relevance value.

    v0.1 heuristic: if relevance > 0.8 and abstraction is shorter than 30 chars, warn.
    """
    warnings: list[str] = []
    if relevance > 0.8 and len(abstraction) < 30:
        warnings.append(
            f"high relevance ({relevance:.2f}) but short abstraction ({len(abstraction)} chars)"
        )
    return warnings

def distinguishability_test(abstraction: str) -> list[str]:
    """SOFT WARN: return list of warning strings if abstraction is so generic
    that distinct source documents would produce equal abstractions.

    v0.1 heuristic: if abstraction is a known generic template phrase (exact match),
    warn. Phase 4 may extend with corpus-statistical checks.
    """
    warnings: list[str] = []
    GENERIC_TEMPLATES = {
        "document of length class short, topic category general, temporal current",
        "conversation with multiple participants, topic domain general, tone neutral",
    }
    if abstraction in GENERIC_TEMPLATES:
        # Spec note: the templated v0.1 strategies ARE generic by design;
        # warning is informational, not blocking.
        warnings.append(f"abstraction matches a known generic template; consider adding corpus-statistical signals")
    return warnings
```

**`src/photophore/shadow/_generate.py`** — generate() main entry point:
```python
"""generate() — main shadow generation entry point per SHADOW-01..06.

Order of operations:
  1. _generate_abstraction(content, content_type) — per-type strategy
  2. irreversibility_test() — HARD FAIL raises ShadowIrreversibilityError
  3. relevance_preservation_test() + distinguishability_test() — SOFT WARN -> warnings list
  4. shadow_id = str(uuid.uuid4()) — ALWAYS fresh per call (SHADOW-01, SHADOW-02, SHADOW-06)
  5. Return ShadowResult(shadow, warnings)

NO CACHING: no module-level shadow cache, no @lru_cache, no @functools.cache.
SHADOW-06 verified by grep gate AND by behavioral test (100 identical-input calls -> 100 distinct shadow_ids).
"""
from __future__ import annotations
import uuid
from ._quality import (
    distinguishability_test,
    irreversibility_test,
    relevance_preservation_test,
)
from ._strategies import _generate_abstraction
from ._types import ContentType, Shadow, ShadowResult

def generate(content: bytes, content_type: ContentType, relevance: float = 0.5) -> ShadowResult:
    """Generate a fresh shadow per SHADOW-01..06.

    Args:
        content: source content bytes (NEVER stored or cached).
        content_type: ContentType enum (one of the 6 spec-mandated categories).
        relevance: float in [0.0, 1.0]; declared relevance to the dispatch.

    Returns:
        ShadowResult(shadow, warnings) where:
          - shadow: Shadow with fresh UUIDv4 shadow_id, tier=1.
          - warnings: tuple of soft-fail diagnostic strings (may be empty).

    Raises:
        ShadowIrreversibilityError: if the abstraction leaks any source substring
        of length >= 8 (HARD FAIL — dispatch must abort).
    """
    # Step 1: abstraction (per-type strategy)
    abstraction = _generate_abstraction(content, content_type)

    # Step 2: irreversibility (HARD FAIL)
    irreversibility_test(content, abstraction)  # raises ShadowIrreversibilityError on leak

    # Step 3: soft-warn quality tests
    warnings: list[str] = []
    warnings.extend(relevance_preservation_test(content, abstraction, relevance))
    warnings.extend(distinguishability_test(abstraction))

    # Step 4: fresh UUIDv4 shadow_id — SHADOW-01, SHADOW-02, SHADOW-06.
    # NEVER cached. Each call produces a fresh ID, even for identical inputs.
    shadow_id = str(uuid.uuid4())

    shadow = Shadow(
        shadow_id=shadow_id,
        content_type=content_type,
        abstraction=abstraction,
        relevance=relevance,
        tier=1,
    )
    return ShadowResult(shadow=shadow, warnings=tuple(warnings))
```

    POLICY-02: derived from channel.ceiling + envelope output_contract.

    """Return True if received_result respects authored_policy; False if it violates.

    POLICY-03: full integration is Phase 3 (dispatch coordinator step 9 — verify-receipt
    then compare-result). Phase 2 ships the comparison helper; Phase 3 wires it.

    v0.1 semantics:
      - If `strip_before_persist` lists field "*", the received_result MUST NOT contain
        any field intended for persistence — violation if any persist-candidate field is present.
      - If `strip_before_persist` lists specific fields, those fields MUST NOT be in the
        persisted-result subset — violation if any listed field is present.
      - If `return_only` is non-empty, ALL fields in received_result MUST be in the union of
        return_only + persist_to_shared.
      - If `persist_to_shared` is non-empty, fields in received_result intended for shared-store
        MUST be a subset of persist_to_shared.
    """
    persisted_fields = set(received_result.get("persisted_fields", []))
    returned_fields = set(received_result.get("returned_fields", []))

    if "*" in authored_policy.strip_before_persist:
        if persisted_fields:
            return False  # tier-0: nothing should persist

    for stripped in authored_policy.strip_before_persist:
        if stripped == "*":
            continue
        if stripped in persisted_fields:
            return False  # specific stripped field appeared in persisted

    if authored_policy.return_only:
        allowed = set(authored_policy.return_only) | set(authored_policy.persist_to_shared)
        if returned_fields - allowed:
            return False  # returned a field outside the allowed union

    if authored_policy.persist_to_shared:
        allowed_persist = set(authored_policy.persist_to_shared)
        if persisted_fields - allowed_persist:
            return False  # persisted a field outside the allowed list

    return True
```

"""`photophore policy` CLI subcommand group. CLI-05: policy preview."""
from __future__ import annotations
import json
from pathlib import Path
import click
from ..audit import AuditLog
from ..channels import ChannelStore
from ..core import ChannelId
from ..policy import author, ResultPolicy
from ._errors import KeystoreError
from ._format import emit_json_document

@click.group("policy")
def policy() -> None:
    """Result policy authoring (preview / future: edit)."""

@policy.command("preview")
@click.option("--channel", "channel_id", required=True, help="Channel id (UUIDv4) to author policy for.")
@click.option("--task", "task_path", required=True, type=click.Path(exists=True, path_type=Path),
              help="Path to task envelope draft JSON.")
@click.pass_context
def preview(ctx: click.Context, channel_id: str, task_path: Path) -> None:
    """Show the result_policy that would be authored for a given channel + envelope draft.

    POLICY-01: any result_policy field in the draft is IGNORED. The output is the
    authored policy, not the draft's policy.
    """
    audit_db = ctx.obj["audit_db"]
    channels_db = ctx.obj["channels_db"]
    audit_log = AuditLog(audit_db)
    store = ChannelStore(channels_db, audit_log)
    try:
        channel = store.show(ChannelId(channel_id))
    except Exception as exc:
        raise KeystoreError(f"channel {channel_id!r} not found in keystore: {exc}") from exc
    draft = json.loads(task_path.read_text())
    authored = author(channel, draft)
    if ctx.obj.get("json"):
        emit_json_document({
            "channel_id": channel.id,
            "channel_ceiling": channel.ceiling,
            "authored_policy": {
                "persist_to_shared": authored.persist_to_shared,
                "return_only": authored.return_only,
                "strip_before_persist": authored.strip_before_persist,
            },
            "draft_policy_ignored": draft.get("result_policy") is not None,
        })
    else:
        click.echo(f"channel: {channel.id}  ceiling: {channel.ceiling}")
        click.echo(f"authored result_policy:")
        click.echo(f"  persist_to_shared:    {authored.persist_to_shared}")
        click.echo(f"  return_only:          {authored.return_only}")
        click.echo(f"  strip_before_persist: {authored.strip_before_persist}")
        if draft.get("result_policy") is not None:
            click.echo(f"(NOTE: draft contained a result_policy field; it was IGNORED per POLICY-01.)")
```
